Terminus build tools page on Pantheon
https://github.com/pantheon-systems/terminus-build-tools-plugin

mkdir -p ~/.terminus/plugins
composer create-project -d ~/.terminus/plugins pantheon-systems/terminus-build-tools-plugin:~1

Build new site:
terminus build-env:create-project --team="Awesome Agency" d8 demo4-11-2017

Create token:

https://circleci.com/account/api
